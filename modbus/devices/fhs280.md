# Modbus Protocol

| Parameter                | Value                                            |
| ------------------------ | ------------------------------------------------ |
| Mode                     | RTU RS485                                        |
| Baud                     | Configurable (setpoint register)                 |
| Start bits               | 1                                                |
| Data bits                | 8                                                |
| Stop bits                | 1 stop bit (Even/Odd), 2 stop bits (None parity) |
| Parity                   | Configurable                                     |
| Address                  | Configurable                                     |
| Supported Function Codes | 3, 4, 6, 17                                      |
